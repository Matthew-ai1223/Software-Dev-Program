# How to Start the Project

This project uses a PHP backend and needs a local server (XAMPP) to run.

## 1. Start Server
1. Open **XAMPP Control Panel**.
2. Click **Start** for **Apache**.
3. Click **Start** for **MySQL**.

## 2. Access the App
Open your web browser and click this link:
[http://localhost/new%20files/3-month/reg.html](http://localhost/new%20files/3-month/reg.html)

> **Note:** Do not open the `.html` file directly (e.g. `file:///C:/...`) or the backend connection will fail. You must use `localhost`.

## 3. TroubleShooting
If the database isn't working, check the status here:
[http://localhost/new%20files/3-month/backend/test_connection.php](http://localhost/new%20files/3-month/backend/test_connection.php)
